# Netdata-Rpi
Netdata for Raspberry Pi with minimal dashboard.  
Build image and run : `./netdata.sh`  
Browse `http://your-pi-IP:19999/rpi.html`  
## Options of netdata.sh
`-h` Help  
`-p` Pull latest Netdata image and build
